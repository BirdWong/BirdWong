CLion调试redis源码

背景

CLion使用CMake来管理编译，而redis源码本身使用make，因此直接将redis源码导入CLion无法直接运行，需要配置CMake。

由于学习过程中参考的书籍为《Redis 设计与实现》，因此源码版本也跟本书保持一致

步骤

  1. 下载源码

  2. deps/hiredis目录下新增CMakeLists.txt

  3. deps/linenoise目录下新增CMakeLists.txt

  4. deps/lua目录下新增CMakeLists.txt

  5. deps目录下新增CMakeLists.txt

  6. redis-3.0-annotated目录下新增CMakeLists.txt5

  1. 导入CLion

    将项目导入CLion，导入时选择不覆盖已有的CMakeLists.txt。

  2. 运行
    在CLion中选择redis-server，选择运行/调试，即可成功运行。
